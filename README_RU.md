**ProfanityMute** — утилита для стримеров, видеомонтажёров и авторов контента, которые монтируют записи стримов и нарезки. Программа автоматически распознаёт речь, находит нецензурную лексику с точностью до миллисекунд, глушит или приглушает нежелательные слова и экспортирует маркеры прямо на таймлайн в DaVinci Resolve.

В отличие от стандартного софта, который перекодирует весь видеофайл часами, ProfanityMute использует прямой поток FFmpeg (Direct Stream Copy). Видеодорожка не затрагивается вообще, поэтому 2-часовой файл 1080p собирается за несколько секунд после распознавания речи.

---

## Ключевые возможности

* **Прямой поток FFmpeg (Direct Stream Copy)**: Видеопоток копируется без передекодирования. Обрабатывается только аудиодорожка, которая затем вшивается обратно в контейнер (.mp4, .mov, .mkv).
* **Глушение только корня слова (Root-Only Muting)**: Вместо вырезания слова целиком (*"за-е-бал-ся"*), алгоритм вычисляет временные границы матерного корня (*"за...ался"*). Приставки и окончания остаются слышимыми, создавая эффект аккуратного ручного монтажа для YouTube.
* **Маркеры для DaVinci Resolve**: Автоматический экспорт маркерных файлов `.edl` (со смещением `01:00:00:00` под дефолтный стартовый таймкод DaVinci) и `.csv`. Маркеры накладываются прямо на существующий таймлайн в 2 клика.
* **Поддержка специализированной русской модели**: Интеграция с дообученной моделью `antony66/whisper-large-v3-russian` (через CTranslate2 сборку `bzikst/faster-whisper-large-v3-russian`), которая точнее распознаёт русскую быструю речь, геймерский сленг и мат на стримах.
* **Работа с видео и отдельным аудио**: Принимает видеозаписи (`.mp4`, `.mov`, `.mkv`, `.avi`), а также отдельные звуковые дорожки (`.wav`, `.mp3`, `.m4a`, `.flac`, `.aac`).
* **Матометр и Инспектор транскрибации**: Встроенный интерфейс с графиком плотности мата по минутам и поисковой таблицей транскрибации текста с подсветкой матерных слов для быстрой проверки точности.
* **GPU-ускорение CUDA**: Нативная поддержка NVIDIA CUDA под Windows (`RTX/GTX`) с автоматическим подключением необходимых DLL-библиотек.
* **Авто-папка (Watch Folder)**: Фоновый мониторинг папки с записями OBS или Streamlabs с автоматическим запуском цензурирования после завершения трансляции.

## Установка

### Требования
* **ОС**: Windows 10 / 11, Linux или macOS.
* **Python**: 3.10 или 3.11.
* **FFmpeg**: Установлен и доступен в системной переменной PATH.

### 1. Клонирование репозитория и установка зависимостей

```bash
git clone https://github.com/JustPacer/profanity-mute.git
cd profanity-mute
pip install -r requirements.txt
```

### 2. Включение GPU-ускорения NVIDIA CUDA (Для видеокарт RTX/GTX)

```bash
pip install nvidia-cublas-cu12 nvidia-cudnn-cu12
```

## Быстрый запуск

### Графический интерфейс (GUI)

Запуск графического интерфейса:

```bash
python main.py
```

Возможности GUI:
* Перетаскивание видео и аудиофайлов (`.mp4`, `.mov`, `.wav`, `.mp3`).
* Переключение режима глушения только корня слова и экспорта маркеров.
* Выбор типа глушения (Приглушение -24dB, Mute или Запикивание Beep).
* График плотности мата и инспектор расшифровки речи с поиском.
* Настройка фонового мониторинга авто-папки.

### Консольный интерфейс (CLI)

Обработка файла с приглушением мата на -24 dB:

```bash
python main.py my_stream.mp4 --mode volume_ducking --db -24 --model antony66/whisper-large-v3-russian
```

Вывод всей накопительной статистики за всё время:

```bash
python main.py --stats
```

Запуск фонового мониторинга папки с записями OBS:

```bash
python main.py --watch-dir "C:\Streams\OBS"
```

## Импорт маркеров в DaVinci Resolve

1. Обработайте файл в ProfanityMute.
2. Откройте проект и нужный таймлайн в **DaVinci Resolve**.
3. В панели **Media Pool** (слева вверху) нажмите правой кнопкой мыши по значку вашего **Таймлайна**.
4. Выберите **Timelines ➔ Import ➔ Timeline Markers from EDL...**.
5. Укажите сгенерированный файл `*_davinci_markers.edl`.
6. Красные маркеры с подписями вырезанных слов появятся прямо на вашем таймлайне.
