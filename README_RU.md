# 🔇 ProfanityMute (AutoCens) — Автоматическое запикивание и приглушение мата в видео и аудио

ПО для контент-мейкеров, стримеров и монтажёров, позволяющее в автоматическом режиме находить нецензурную лексику в видео и аудиозаписях любой длины (включая стримы более 2 часов в 1080p), приглушать её и выводить накопительную статистику.

[English Documentation (README.md)](README.md) | [Подробная инструкция по установке (INSTALLATION_GUIDE.md)](INSTALLATION_GUIDE.md)

## ✨ Ключевые возможности

- **⚡ Без потери качества и мгновенный экспорт (FFmpeg Direct Stream Copy)**: Видеопоток НЕ перекодируется. Программа заменяет только обработанный звук. 2-часовой файл 1080p собирается за считанные секунды после окончания распознавания.
- **✂️ Глушение ТОЛЬКО корня слова (Root-Only Muting)**: Вместо глушения всего слова целиком (*"за-е-бал-ся"*), программа вычисляет границы корня и глушит **только сам матерный корень** (*"за...ался"*). Приставки и окончания остаются слышимыми, создавая эффект ручного монтажа.
- **📌 Авто-экспорт маркеров для DaVinci Resolve**: Автоматическая генерация файлов `*_davinci_markers.edl` (с таймкод-смещением `01:00:00:00` под стандартные таймлайны DaVinci) и `*_davinci_markers.csv`.
- **🎵 Прямая поддержка аудио и видеофайлов**: Работает с видеозаписями (`.mp4`, `.mov`, `.mkv`, `.avi`) и отдельными звуковыми дорожками (`.wav`, `.mp3`, `.m4a`, `.aac`, `.flac`).
- **🚀 Аппаратное ускорение NVIDIA CUDA**: Ускорение обработки в 5–8 раз на видеокартах NVIDIA (`RTX / GTX`) с авторегистрацией DLL в Windows.
- **🔥 Детектор эмоциональных пиков (Hype Moments)**: Поиск участков с высокой плотностью мата для моментального обнаружения лучших клипов под YouTube Shorts / TikTok / Reels.
- **📁 Переключаемая Авто-папка (Watch Folder)**: Автоматический мониторинг папки записей OBS/Streamlabs и цензурирование свежих стримов в фоновом режиме.
- **📊 Накопительная статистика мата**: Хранение базы данных (`data/stats.json`) со счётчиком матов за все запуски (например, до конца года) и частотным словарем слов.
- **🔊 3 Режима глушения**:
  - `Приглушение (Volume Ducking -24dB)` *(Рекомендуется для монтажа)*
  - `Полная тишина (Mute 0dB)`
  - `Классический писк (Beep 1000Hz)`

## 🛠 Быстрая установка

### 1. Клонирование репозитория и установка зависимостей:

```bash
git clone https://github.com/JustPacer/profanity-mute.git
cd profanity-mute
pip install -r requirements.txt
```

### 2. Включение GPU-ускорения NVIDIA CUDA (для владельцев RTX/GTX):

```bash
pip install nvidia-cublas-cu12 nvidia-cudnn-cu12
```

## 🚀 Запуск

### Графический интерфейс (GUI)

```bash
python main.py
```

В GUI доступно:
- Перетаскивание файлов (`.mp4`, `.mov`, `.mkv`, `.wav`, `.mp3`).
- Тумблер *"✂️ Глушить только корень слова"*.
- Выбор уровня приглушения в dB, точности модели и устройства GPU/CPU.
- Просмотр накопительных карт и таблиц частоты матерных слов.
- Вкладка настройки Авто-папки.

### Консольный интерфейс (CLI)

Приглушить мат в видео или аудиофайле с уровнем -24 dB:

```bash
python main.py my_stream.mp4 --mode volume_ducking --db -24 --model medium
```

Посмотреть всю накопительную статистику за всё время:

```bash
python main.py --stats
```

Запустить фоновый мониторинг папки с видеозаписями:

```bash
python main.py --watch-dir "C:\Streams\OBS"
```

## 🎬 Импорт маркеров в DaVinci Resolve (за 2 клика)

1. Обработайте файл в ProfanityMute.
2. В **DaVinci Resolve** откройте ваш существующий таймлайн.
3. В панели **Media Pool** нажмите **ПКМ по значку вашего Таймлайна**.
4. Выберите **`Timelines` ➔ `Import` ➔ `Timeline Markers from EDL...`**.
5. Укажите сгенерированный файл **`*_davinci_markers.edl`**.
6. Красные маркеры мата мгновенно появятся прямо на вашем таймлайне!
