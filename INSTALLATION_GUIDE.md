# 📖 Инструкция по установке и запуску AutoCens

Подробное руководство по установке и настройке приложения **AutoCens** (автоматическое запикивание мата + генерация маркеров для DaVinci Resolve) на другом ПК.

---

## 💻 Системные требования

- **ОС**: Windows 10 / Windows 11 (64-bit).
- **Видеокарта**: NVIDIA GeForce (серии GTX 10xx, RTX 20xx, 30xx, 40xx) — *желательно для ускорения*, либо мощный процессор (CPU).
- **Диск**: ~3-5 ГБ свободного места (для моделей Whisper и библиотек).

---

## ⚙️ Шаг 1. Установка компонентов (Python)

### 1.1. Установка Python
1. Скачайте [Python 3.10 или 3.11](https://www.python.org/downloads/).
2. При запуске установочного файла **ОБЯЗАТЕЛЬНО поставьте галочку**:
   > 🔴 **`Add Python to PATH`** *(Добавить Python в переменные среды)*.
3. Нажмите **Install Now**.

---

## 🚀 Шаг 2. Скачивание и Установка AutoCens

1. Перенесите папку с проектом `autocens` на нужный компьютер (например, в `C:\Tools\autocens` или `C:\Users\Имя\Documents\autocens`).
2. Откройте командную строку (**cmd**) или **PowerShell** в папке проекта.
   *(Для этого зажмите `Shift` + ПКМ по пустой области папки ➔ "Открыть окно PowerShell здесь")*.

3. Выполните установку основных библиотек одной командой:
   ```bash
   pip install -r requirements.txt
   ```

4. **Для ПК с видеокартой NVIDIA** (для ускорения в 5-8 раз на CUDA):
   ```bash
   pip install nvidia-cublas-cu12 nvidia-cudnn-cu12
   ```

---

## 🎬 Шаг 3. Первый запуск приложения

### Вариант 1. Графический интерфейс (GUI)
В командной строке в папке проекта выполните:
```bash
python main.py
```
*(Или создайте ярлык файла `main.py` на рабочем столе)*.

### Вариант 2. Консольный режим (CLI / Скрипты)
```bash
python main.py "C:\Streams\my_video.mp4" --mode volume_ducking --db -24 --model medium
```

---

## 🛠 Руководство по работе с софтом

### 1. Режимы глушения (Выборка):
- **Приглушение (Volume Ducking -24dB)** *(Рекомендуется)*: Заглушает мат до тихих -24 дБ, сохраняя атмосферу стрима. Идеально под монтаж.
- **Mute**: Полная тишина на месте матерного слова.
- **Beep**: Классический писк (1000 Гц).

### 2. Импорт маркеров в DaVinci Resolve:
1. После обработки видео рядом с ним автоматически появится файл: `*_davinci_markers.csv`.
2. В **DaVinci Resolve** откройте проект и выберите в меню:
   > **File ➔ Import ➔ Timeline Markers...**
3. Укажите сгенерированный `.csv` файл.
4. На вашем таймлайне появятся **красные маркеры** на каждом матерном слове с подписью произнесенной фразы.

### 3. Настройка Авто-папки (Watch Folder):
1. Перейдите во вкладку **`📁 Авто-папка (Watch Folder)`**.
2. Укажите папку, куда OBS сохраняет видеозаписи.
3. Включите тумблер **`🟢 Включить фоновый мониторинг`**.
4. Теперь софт будет сам автоматически обрабатывать свежие записи после стрима!

## ❓ Часто задаваемые вопросы и решения проблем

#### ❌ Проблема: *"Python не распознается как внутренняя команда"*
* **Решение**: Вы не поставили галочку `Add Python to PATH` при установке Python. Переустановите Python, поставив эту галочку.

#### ❌ Проблема: *Крякозябры (РњР°С‚) в маркерах DaVinci*
* **Решение**: В последней версии AutoCens файлы создаются в кодировке `UTF-8 с BOM (utf-8-sig)`, что гарантирует корректное отображение русского языка.
